## The Security History (The Honest Version)

In September 2023, CoinEx suffered a hot wallet hack that resulted in roughly $70 million in losses. This is a real data point and worth knowing.

What followed matters too: no customer funds were lost. The exchange covered the losses internally using operational reserves. Since then, CoinEx has directed 10% of all trading fees generated into a Shield Fund — which reportedly stood at over $33 million by mid-2025 — specifically to cover future incidents.

The response to the 2023 hack — full coverage, no customer losses — is actually the kind of thing that builds trust over time, even if the incident itself is a mark against them.

Current security measures include multi-signature cold wallet storage for the majority of user assets, two-factor authentication, withdrawal whitelisting, and anti-phishing codes.
